# Advanced Polls Plugin

Richer poll modes beyond single questions.

## New behaviors
- **Poll series**: send multiple polls back-to-back and collect all results before summarising
- **Anonymous tally**: report counts only, never name voters — e.g. "3 votes for Yes"
- **Ranked choice**: ask players to DM their ranking 1-by-1

## Host DM commands
- "run poll series: [Q1] / [Q2] / [Q3]" → sends polls sequentially
- "anonymous poll: [question] / [opt1] / [opt2]" → hides voter names in results
