# Scorekeeper Plugin

Track cumulative scores across rounds.

## New behaviors
- Maintain a named scoreboard in context across phases
- Post updated leaderboard after each scoring event
- Accept "+N name" shorthand from host to award points

## Host DM commands
- "+5 Alice" → award 5 points to Alice
- "scoreboard" → post current standings to group
- "reset scores" → zero out all scores
